# 习题 4.2.4
我们可以用等式说一个函数对于输入应该如何计算，可以用等式说明手工计算表达式的过程，也可以将添加在DrRacket的定义窗口中以测试程序例子。  
使用断言重新表示习题2.2.1、2.2.2、2.2.3和2.2.4中的例子。  
方法一：
```
;; 我们的函数
(define (Fahrenheit->Celsius f)
  (/ (* (- f 32) 5) 9))
;; 测试表达式
（Fahrenheit->Celsius 32)
;; 预期值：
0

```
输出如下：
```
0
0
```
方法二：
```
;; 我们的函数
(define (Fahrenheit->Celsius f)
  (/ (* (- f 32) 5) 9))
;; 写成断言：
(= (Fahrenheit->Celsius 212) 100)
```
输出如下：
```
#true
```
