# 习题4.3.3
试确定当n为(a)500 (b)2800 和(c)15000时下述表达式的值。
```
  (cond
    [(<= n 1000)(* .040 1000)]
    [(<= n 5000)(+ (* 1000 .040)
                    (* (- n 1000) .045))]
    [else(+ (* 1000 .040)
             (* 4000 .045)
             (* (- n 10000) .055))]))
```
答：
```
(define (A n)
  (cond
    [(<= n 1000)(* .040 1000)]
    [(<= n 5000)(+ (* 1000 .040)
                    (* (- n 1000) .045))]
    [else(+ (* 1000 .040)
             (* 4000 .045)
             (* (- n 10000) .055))]))
(A 500)
(A 2800)
(A 15000)
```
输出如下：
```
40
121
495
```
