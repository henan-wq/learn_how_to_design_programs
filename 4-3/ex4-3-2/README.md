# 习题4.3.2
试确定当n为(a)500 (b)2800 和(c)15000时下述表达式的值。
```
(cond
  [(<= n 1000) .040]
  [(<= n 5000) .045]
  [(<= n 10000) .055]
  [(> n 10000) .060]))
```
答：
```
(define (A n)
  (cond
    [(<= n 1000) .040]
    [(<= n 5000) .045]
    [(<= n 10000) .055]
    [(> n 10000) .060]))
(A 500)
(A 2800)
(A 15000)
```
输出如下：
```
0.04
0.045
0.06
```
