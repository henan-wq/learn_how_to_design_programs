# 习题2.1.1
查明DrRacket是否具备平方、计算一个角度的正弦值以及确定两个数的最大值的运算。  
  
答：  
平方：  
```
> (sqr 2)
4
```
角度的正弦值：  
```
> (sin 90)
#i0.8939966636005579
> (sin 80)
#i-0.9938886539233752
> (sin 0)
0
> (sin 30)
#i-0.9880316240928618
> (sin 60)
#i-0.3048106211022167
```
确定两个数的最大值的运算：
```
> (> 30 70)
#false
> (< 30 70)
#true
> (= 30 70)
#false
```
